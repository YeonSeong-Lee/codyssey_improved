# Codyssey Improved

Codyssey Improved는 Codyssey 플랫폼의 편의성을 개선하기 위한 크롬 확장 프로그램입니다.

## 주요 기능

- ✨ 드래그로 간편한 시간 선택
- 🎯 한 번에 여러 시간대 평가 요청
- 📅 목차 기본 숨기기 기능
- 🔄 평가 요청 후 자동 새로고침
- 🔄 평가 요청 전체 취소
- 🔄 평가 요청 선택 취소

## 사용 방법

1. Codyssey 평가 요청 페이지 접속
2. 원하는 시간대를 마우스로 드래그하여 선택
3. 여러 시간대 선택 가능
4. '평가 하기' 버튼 클릭
5. '선택된 셀 지우기' 버튼으로 선택 초기화
6. '평가 전체 취소' 버튼으로 선택한 모든 평가를 취소
