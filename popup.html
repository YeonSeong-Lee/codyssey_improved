Codyssey Improved 소개

Codyssey Improved는 Codyssey 사이트에서 편의성을 위한 크롬 확장 프로그램입니다.

주요 기능

  • 드래그로 간편한 평가 요청
  • 목차 기본 숨기기

사용 방법

  1. 원하는 시간대를 마우스로 드래그하여 선택합니다.
  2. 여러 시간대를 선택할 수 있습니다.
  3. '평가 요청하기' 버튼을 클릭하면 선택한 모든 시간대에 대해 평가가 요청됩니다.
  4. '시간 다 지우기' 버튼으로 선택을 초기화할 수 있습니다.
* 이 확장 프로그램은 Codyssey 시스템을 개선하여 사용자 경험을 향상시키기 위해 제작되었습니다.
